# vrmh
VR microcontroller haptics

## You need

 - OSC avatar
 - ESP01-S + buck converter / WEMOS D1 (fits in frunk) / ESP8266 / etc...
 - soldering iron
 - [For index frunk](https://www.aliexpress.com/item/32793566256.html)
 - Phone vibration motor
 - Velcro tape
 - Arduino IDE / VSCode
 - Python 3.10+
 - Extreme patience
